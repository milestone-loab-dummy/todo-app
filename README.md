# Todo App

A full-stack todo application built with:

- **Frontend**: Next.js 14 · React · TypeScript · Tailwind CSS — `localhost:3000`
- **Backend**: Go · Gin · GORM · SQLite — `localhost:8080`

## Getting started

### Backend

```bash
cd backend
go run ./...
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Running tests

```bash
cd backend && go test ./...
```
